# FetchTest

Description:
- Single activity app with one screen, that shows list of items from JSON

Stack:
 - Kotlin only app
 - Coroutines for asynchronous calls
 - Koin as DI
 - MVVM with MVI 
 - Clean Code
 - Jetpack Compose as UI approach
 - Retrofit with OkHttp for requests 
 - Moshi for parse JSON into objects
